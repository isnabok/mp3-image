## Frequently Asked Questions

### What files are supported?

The editor is designed for MP3 files.

### Can I change the cover image?

Yes. You can upload a new `PNG`, `JPG`, or `WEBP` image, and the artwork will be optimized before it is embedded into the MP3 file.

### Can I edit the filename before downloading?

Yes. You can choose a new output filename before saving the updated file.

### Do I need to install anything?

No. The project is designed to work through the web interface without a separate desktop installation flow.

### Which metadata fields can I edit?

You can update the most common ID3 fields, including:

- title
- artist
- album
- genre
- year
- track
- comment
- cover image
