## Privacy Policy

This page explains the general privacy approach for MP3 Tag Editor.

## Uploaded Files

When you upload an MP3 file or a cover image, the file is processed so metadata and artwork can be read or updated. The project is intended to handle only the file operations needed for that editing flow.

## Metadata Processing

The editor may read information such as:

- filename
- title
- artist
- album
- genre
- year
- track
- comment
- embedded cover art

This information is used only to display and update the file content within the editor flow.

## Technical Information

Like most websites and applications, the service may temporarily process technical request data required for normal operation, security, debugging, or performance monitoring.

## Third-Party Services

If analytics, error tracking, hosting providers, or similar infrastructure are used, their handling of technical data may also apply according to their own policies.

## Changes To This Policy

This policy may be updated over time as the project changes.
