Use batch mode when you want to load several MP3 files into one queue and move through them card by card. Each file keeps its own metadata form, audio preview, cover controls, and save action, which makes the workflow easier to review than repeatedly reopening the single-file editor.

## What Batch Mode Does

- load multiple MP3 files into one queue
- open each track in its own expandable editing card
- update title, artist, album, genre, year, track, comment, and output filename independently
- replace or remove cover art for a specific file
- save and download each updated MP3 separately

## Why Use It

Batch mode is useful when you are cleaning up a small collection, reviewing a release, or stepping through several files from the same source. The editing state stays separated by card, so you can focus on one track at a time without losing the context of the full queue.

## Current Scope

The current version is designed around queue-based editing and per-file downloads. It is a practical middle step between the single-file editor and a future bulk-export flow, which means the experience stays controlled and predictable while still removing repetitive upload steps.

## Frequently Asked Questions

### What is batch MP3 editing?

Batch MP3 editing means working with several MP3 files in one session instead of opening them one by one. Each file can keep its own metadata fields, cover controls, and save action while still being part of the same queue.

### Can I save all files at once in batch mode?

The current batch version is focused on queue-based editing with individual save actions for each file. This keeps the workflow simple and makes it easier to review every track before downloading the updated MP3.
