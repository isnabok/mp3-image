## A Focused MP3 Metadata Workflow

MP3 Tag Editor is designed for one job: take an MP3 file, inspect the current metadata, make the changes you need, and export a cleaned-up version without jumping through extra tools or heavy desktop software.

## What You Can Do

- update common ID3 fields such as title, artist, album, genre, year, track, and comment
- rename the output file before download
- preview the current embedded cover art
- upload a new image for the cover
- download the updated MP3 immediately after saving

## Why This Setup Works Well

The project keeps the application part and the content part separate:

- the editor stays focused on file processing
- SEO content can be expanded with MDX
- static pages can be added without a database
- content can be versioned directly in git

## Frequently Asked Questions

### What can I edit in an MP3 file?

You can update the filename, title, artist, album, genre, year, track, comment, and cover art.

### Can I replace the cover image?

Yes. You can upload a new `PNG`, `JPG`, or `WEBP` image, and it will be prepared for embedding into the MP3 file.

### Do I need a database or account?

No. The site is intentionally built around file-based content and a direct editing workflow.
